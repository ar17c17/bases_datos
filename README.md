# Práctica de Bases de Datos - Gestión de Emisiones y Catálogo Musical

Este repositorio contiene el desarrollo completo de la práctica de Bases de Datos, que abarca desde el diseño del esquema relacional hasta el mantenimiento y consulta avanzada de los datos.

## Estructura del Proyecto
* **`practica.sql`**: Definición de la estructura de la base de datos (tablas, claves primarias/foráneas y restricciones). Datos iniciales para la población de las tablas.
* **`Practica2_3.sql`**: Consultas SQL avanzadas, uso de JOINs, subconsultas y creación de Vistas.
* **`Practica4.sql`**: Operaciones de mantenimiento, actualización, borrado y modificación del esquema.
